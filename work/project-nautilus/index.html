Project Nautilus EST. MMXXIV  —  DISPATCHES FROM WIDOW'S BAY  —  AN ONGOING INVESTIGATION

A seal that looks like it was found, not designed. A brand that exists because one person keeps pulling the thread.

Project Nautilus is a personal world-building project — a multi-platform mystery built around a fictional investigative journalist named E. Crane, a journal with a false bottom, and a family secret that refuses to stay buried. Every element of the brand is designed to feel discovered rather than made: the wax-seal mark, the found-document type system, the palette that reads like something left out in the rain.

I built the full brand identity from scratch — mark, color system, voice guidelines, and a five-typeface hierarchy that maps to how the story talks. The constraint: nothing should look designed. Everything should look like evidence.

01 // The Mark
N nautilus
Project Nautilus
E. Crane

The Chambered Seal.

A nautilus shell in cross-section, set inside a collector's stamp. It reads as a wax seal, a coin, a watermark — the kind of mark that shows up on the evidence, not on a billboard. The chambers spiral inward: every answer opens a smaller room.

The mark is built for a single constraint: it has to look like it predates the project. It can't look like a logo. It has to look like something E. Crane found in the box alongside the journal — an emblem belonging to the family he's investigating.

Single color only — Gilt on dark, Abyss on bone. Never gradients.
Clear space = the height of one chamber on all sides.
Below 24px, drop the circular text. Use the shell alone.

04 // Voice & Tone

It's real. I'm scared. Read it with me. Every word, everywhere — Substack, Patreon, email, captions, replies — is written by one person who believes this is happening to him. The illusion is the product.

Urgent
Every word earned. Nothing decorative. Say it and move — there isn't time.
"Grab a copy of the file. I don't know how long these will stay up."
Weathered
Nothing is pristine. The brand has been through something, and it shows in every torn edge.
"This page survived. Barely."
Collaborative
The reader is a witness. Ask them to dig. Leave threads open. Never withhold.
"Am I seeing this right? Someone please tell me I'm wrong."
Doubtful
A skeptic being slowly convinced. Hedge. Second-guess. Show your work. The dread is in the dates.
"Probably nothing. But if the next dispatch is late, it isn't because I stopped looking."
